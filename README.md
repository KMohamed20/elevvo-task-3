# Task 3: One-Page Website for TaskFlow

> **Level**: Level 1  
> **Requirements Met**: ✅ Hero section | ✅ Features | ✅ Testimonials | ✅ Pricing | ✅ Footer | ✅ Responsive  

## 🔍 Description

A complete, modern one-page marketing website for **TaskFlow** — a fictional productivity app. Built with pure **HTML, CSS, and JavaScript**, it showcases clean design principles and responsiveness.

Sections include:
- Hero with CTA button
- 3 feature cards with icons
- 2 user testimonials
- 3 pricing tiers (Free, Pro, Team) — “Pro” highlighted
- Footer with social links

## 🖥️ Features

- ✅ **Animated Scroll Effects**: Sections fade in as you scroll down.
- ✅ **Responsive Grids**: Cards and pricing adapt to screen size.
- ✅ **Modern UI**: Soft shadows, rounded corners, gradient buttons.
- ✅ **Clear Hierarchy**: Visual emphasis on "Most Popular" plan.
- ✅ **Mobile-First**: Fully functional on phones.
